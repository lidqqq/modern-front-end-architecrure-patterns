# modern-front-end-architecrure-patterns

## はじめに

モダンな技術 stack で真面目に front-end application architecure に取り組んだ場合の実装 sample.

モダンな front-end application では非常に複雑な構成を求められるため, architecure/design pattern もいくつかのものを複合している.

## main stacks

- Next.js
- Redux
